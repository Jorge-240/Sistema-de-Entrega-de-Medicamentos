# Sistema de Entrega de Medicamentos

sistema basico para entrega de medicamentos a pacientes con enfermedades
no transmisibles.

Este sistema mostrar la cantidad de pacientes atendidos, la cantidad de pacientes a los que
se les hizo entrega del medicamento 1 junto al porcentaje de estos respecto al total de pacientes
atendidos formateado a 2 cifras decimales y la cantidad de pacientes a los que se les hizo entrega
del medicamento 2 junto al porcentaje de estos respecto al total de pacientes atendidos
formateado a 2 cifras decimales.
